# MkDocs docs folder for Reflow

This archive contains only the `docs/` folder in a MkDocs-friendly Markdown
layout.

## Included

- Markdown source files for the main documentation
- SVG architecture diagrams under `assets/`
- `mkdocs.yml.example` that you can move to the repository root as `mkdocs.yml`

## Typical setup

1. Copy this `docs/` folder into your repository.
2. Move `docs/mkdocs.yml.example` to the repository root as `mkdocs.yml`.
3. Install MkDocs.
4. Run:

```bash
mkdocs serve
```
